# Project Instructions

## Project Description
This project, ScarifGEO, is used by the Scarif team to showcase the data avaliable in the Scarif API service. Currently we have a "boundries-explorer" which shows all the administrative boundries scarif has and a "country-explorer" which gives he ability to load some random data sets like power plants and military bases by country.

The goal is to showcase all the geographical data we have in an easy to use manner and also provide audit data about the data sets to indicate its completenes.

## Code Style
- Use TypeScript for all new files
- Use TSdocs and incline comments where the intention is not clear
- Use maplibre for maps
